## Quick start

Simply plug-in fresco to your existent [wagmi](https://wagmi.sh) project. It's as easy as:

```bash
npm i @radish-la/fresco
# or
pnpm i @radish-la/fresco
# or
yarn add @radish-la/fresco
```
